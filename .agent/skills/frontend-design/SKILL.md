# Habilidad: Diseño Frontend (Artesano Digital)

Esta habilidad transforma al agente en un diseñador de interfaces de alto nivel, capaz de crear experiencias web que se sienten premium, deliberadas y únicas, alejándose de los patrones comunes y repetitivos de la generación por IA ("AI slop").

## ¿Cuándo usar esta habilidad?

- Al crear nuevos componentes UI o páginas completas.
- Al rediseñar interfaces existentes para que se vean más modernas y profesionales.
- Cuando el usuario pide que algo se vea "impresionante", "premium" o "de vanguardia".

## Principios de Diseño

1. **Dirección Audaz**: Antes de codificar, elige una estética extrema (ej. minimalismo brutal, caos maximalista, industrial, editorial). No te quedes en el medio.
2. **Tipografía de Impacto**: 
   - **NUNCA** uses fuentes genéricas como Inter, Arial o Roboto (a menos que sea por restricción técnica absoluta).
   - Usa combinaciones de Google Fonts expresivas (ej. *Outfit*, *Playfair Display*, *Syne*).
3. **Movimiento con Propósito**: Implementa micro-interacciones y transiciones que den vida a la interfaz. Usa `framer-motion` para React o transiciones CSS refinadas.
4. **Composición Espacial**: Rompe la cuadrícula de vez en cuando. Usa espacios negativos inesperados o layouts asimétricos.
5. **Texturas y Profundidad**: Añade capas de interés visual usando ruido (noise), gradientes complejos (mesh gradients) o patrones geométricos sutiles.

## Cómo usarla

1. **Planificación Visual**: Describe brevemente la estética elegida antes de escribir el código.
2. **Sistema de Tokens**: Define variables CSS para colores, espaciado y tipografía que sean coherentes con la dirección elegida.
3. **Implementación**: Escribe código limpio, accesible y responsivo.
4. **Pulido**: Añade los detalles finales (hovers, estados de foco, animaciones de entrada).

## Lo que se debe evitar (Anti-patrones)

- **Diseños "Seguros"**: Evita el estilo genérico de "Startup SaaS" con botones azules redondeados y fuentes Inter.
- **Estructuras Predecibles**: No te limites a dashboards de tres columnas estándar.
- **Falta de Intención**: Cada color y cada pixel debe tener una razón de ser.
