# Ejemplo: Hola Mundo

Esta habilidad sirve como referencia sobre cómo estructurar las instrucciones en español.

## ¿Cuándo usar esta habilidad?
- Cuando se está aprendiendo a crear habilidades.
- Como plantilla base para nuevas habilidades.

## Cómo usarla

1. Asegúrate de que el archivo se llame `SKILL.md`.
2. Incluye el frontmatter YAML con el nombre y descripción.
3. Define secciones claras con títulos en español.

### Ejemplo de flujo
Si el usuario dice "Hola", esta habilidad respondería con un saludo formal siguiendo las convenciones del proyecto.
