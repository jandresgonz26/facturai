# Prueba de Verificación

Esta es una habilidad temporal para validar el flujo de creación.

## ¿Cuándo usar esta habilidad?
- Únicamente durante la fase de verificación técnica.

## Cómo usarla
1. Ejecutar una tarea simple de prueba.
2. Confirmar que el agente sigue las instrucciones en español.
